# VAPI-AI-Receptionist
An AI-powered voice receptionist built using Vapi AI, designed to handle real-time phone calls, automate appointment bookings, and manage customer interactions with human-like conversations.  This project demonstrates how to build a fully automated AI receptionist using voice AI, workflow automation, and cloud integrations.
